An AI-powered Telegram bot that instantly identifies rice leaf diseases from a photo and delivers treatment advice in Bangla — built for Bangladeshi farmers.

Try the bot
Scan to open on Telegram

@Ricedisease_bd_bot
How it works
1
Take a photo
Place the rice leaf on white paper and photograph it in good lighting
2
Send to the bot
Open Telegram, find the bot, and send the photo directly
3
AI analyses the image
A trained computer vision model classifies the disease with a confidence score
4
Receive advice in Bangla
Google Gemini generates full treatment guidance — medicines, steps, and prevention

Run it yourself
1
Clone & install
https://github.com/nihal4/Rice-Leaf-Disease-Detection.git cd rice-leaf-disease-bot pip install -r requirements.txt
2
Set environment variables
VariableDescription
TELEGRAM_BOT_TOKENFrom @BotFather on Telegram
GEMINI_API_KEYFrom Google AI Studio
WEBHOOK_URLYour public HTTPS deployment URL
WEBHOOK_SECRETRandom secret, min 16 characters
HF_MODEL_URLDirect link to .onnx model on Hugging Face
CLASS_INDICES_PATHPath to class_indices.json
3
Start the bot
python bot.py
